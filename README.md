# Свадебное приглашение Артема и Виты

## Описание

Интерактивный сайт-приглашение на свадьбу с мини-игрой в гольф, таймингом мероприятия, информацией о локации, дресс-коде и формой для гостей.

## Технологии

- Next.js 14 (Static Export)
- React 18
- TypeScript
- CSS (без внешних библиотек)

## Развёртывание на Vercel

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Импортируйте проект из GitHub/GitLab
3. Vercel автоматически определит Next.js и настроит сборку
4. Или используйте CLI:
   ```bash
   npm i -g vercel
   vercel
   ```

## Настройка Google Form для анкеты

Для того чтобы ответы гостей автоматически записывались в Google Таблицу:

### Шаг 1: Создайте Google Form

1. Перейдите на [forms.google.com](https://forms.google.com)
2. Создайте новую форму с такими вопросами (именно в таком порядке):
   - **Имя и Фамилия** (Short answer, required)
   - **Сможете ли вы присутствовать?** (Multiple choice: Да, обязательно буду! / К сожалению, не смогу / Постараюсь, пока не уверен(а))
   - **Какие алкогольные напитки предпочитаете?** (Checkbox: Игристое / Тихие напитки / Крепкий алкоголь)
   - **Есть ли у вас на что-то аллергия?** (Paragraph)

### Шаг 2: Включите запись ответов в таблицу

1. Нажмите **Ответы** (Responses) вверху формы
2. Нажмите на иконку таблицы Google Sheets (зелёная иконка)
3. Выберите **Создать новую таблицу**
4. Теперь все ответы будут автоматически сохраняться в Google Sheets

### Шаг 3: Получите ссылку на форму для встраивания

1. Нажмите **Отправить** (Send) в правом верхнем углу
2. Выберите вкладку **< >** (Embed HTML)
3. Скопируйте URL из кода iframe

### Шаг 4: Обновите код

Откройте файл `app/components/Survey.tsx` и замените:

```typescript
const GOOGLE_FORM_URL = 'https://docs.google.com/forms/d/e/YOUR_FORM_ID/formResponse'
```

на реальный URL вашей формы (часть между `/d/e/` и `/formResponse`).

**Или** проще - замените весь блок отправки формы на iframe с Google Form:

```tsx
<iframe 
  src="URL_ВАШЕЙ_ФОРМЫ" 
  width="100%" 
  height="800" 
  frameBorder="0"
  style={{ borderRadius: '12px' }}
>
  Загрузка…
</iframe>
```

Где `URL_ВАШЕЙ_ФОРМЫ` - это ссылка из шага 3 (без параметров `embedded=true`, просто основной URL формы).

### Шаг 5: Ссылка на таблицу

1. Откройте Google Sheets с ответами
2. Нажмите **Файл → Опубликовать в интернете**
3. Или просто скопируйте URL таблицы из адресной строки
4. Эту ссылку можно отправлять жениху и невесте для просмотра ответов

## Структура проекта

```
app/
├── components/
│   ├── GolfGame.tsx      # Интерактивная игра в мини-гольф
│   ├── Navigation.tsx    # Навигация по секциям
│   ├── WeddingDetails.tsx # Информация о свадьбе
│   ├── Timeline.tsx      # Тайминг мероприятия
│   ├── Location.tsx      # Как добраться
│   ├── Dresscode.tsx     # Дресс-код
│   ├── Gifts.tsx         # Подарки
│   ├── Contacts.tsx      # Контакты и чаты
│   ├── Survey.tsx        # Анкета для гостей
│   └── Final.tsx         # Финальные слова
├── globals.css           # Глобальные стили
├── layout.tsx            # Layout приложения
└── page.tsx              # Главная страница
```

## Кастомизация

- Замените заглушки ссылок в `Contacts.tsx` на реальные ссылки на чаты
- Замените заглушку Pinterest в `Dresscode.tsx`
- Добавьте ссылку на организатора Элю в `Contacts.tsx`
- Настройте Google Form по инструкции выше
